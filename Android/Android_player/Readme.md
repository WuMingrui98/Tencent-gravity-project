## 安装播放器应用

直接安装player.apk，可以实现对rtmp的拉流。目前的应用版本支持绝大多数安卓手机。

如果有修改应用内容或者生成其他版本的apk，进入player_Decoder.zip使用Andorid Studio 重新生成。
