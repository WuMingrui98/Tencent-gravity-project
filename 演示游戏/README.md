# 圣诞主题小游戏

一个简单的圣诞主题小游戏

## 依赖pygame库

安装pygame库

```sh
pip install pygame
```

## 运行

进入code目录`cd ./code/`

>  运行环境为Linux系统下的Python 3.8.10

```sh
python3 ./main.py
```

游戏启动，然后enjoy
